# Assignment 04: NoSQL Databases with MongoDB and Redis

## Files

- `mongodb_assignment.js` - MongoDB bookstore database, CRUD, aggregations, and explain/index comparison.
- `mongodb_optimization_report.md` - short report template for execution-plan screenshots and findings.
- `redis_assignment.txt` - Redis commands for Strings, Hashes, Lists, Sets, TTL, cache, and bonus leaderboard.
- `redis_notes.md` - Redis explanation notes.
- `screenshots/mongodb/` - place MongoDB screenshots here.
- `screenshots/redis/` - place Redis screenshots here.

## MongoDB Run

```bash
mongosh mongodb_assignment.js
```

Take screenshots of:

- Inserted data count.
- CRUD query results.
- Aggregation outputs.
- Explain plan before index.
- Explain plan after index.

## Redis Run

```bash
redis-cli
```

Run commands from `redis_assignment.txt` and take screenshots of each task section.

## Submission

The requested PDF optimization report can be produced from `mongodb_optimization_report.md` after filling in measured values and adding screenshots.
